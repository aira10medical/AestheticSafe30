# AestheticSafe Risk Widget

Esta app en Streamlit permite estimar de forma lúdica el riesgo quirúrgico estético basado en: Caprini, IMC, Edad y Tabaquismo. Perfecta para médicos o pacientes.

## Para ejecutarlo localmente:
1. Instalá Streamlit:
   pip install streamlit

2. Corré el archivo:
   streamlit run app.py

## Uso sugerido:
- Compartir como link en redes
- Colocar como QR en eventos
- Insertar en tu página web
